# Tracker-web

## Возможноти проекта

1. Добавление/Удаление задач в список
2. Сортировка задач по выполнению
3. Счетчик не выполненых задач

## Установка проекта

```bash
git clone https://github.com/leg0rs/Tracker.git Tracker
cd Tracker
npm run setup
npm run build
```

## Запуск проекта

```bash
npm run start
```
